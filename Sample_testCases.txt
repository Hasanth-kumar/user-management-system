🔐 Step A: Register a New User (Optional)
📍 POST http://localhost:8081/api/auth/register

Request Body (JSON):
{
  "username": "john_doe",
  "email": "john@example.com",
  "password": "secure123"
}
✅ Expected Result:

"User registered successfully!"
📌 Skip this if the user is already registered.
